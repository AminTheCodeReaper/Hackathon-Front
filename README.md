# Hooshmand AI — Persian Intelligence Platform

A modern, animated, and fully responsive landing page for **Hooshmand**, the first Persian AI platform for Iranian businesses. This project was built for a hackathon challenge to redesign UI/UX with smooth animations while preserving original content and structure.

## 🚀 Features

- **Fully Responsive** — Works perfectly on mobile, tablet, and desktop
- **Glassmorphism Design** — Modern blurred cards with neon glow effects
- **Smooth Animations** — Scroll-triggered entrances, hover effects, and count-up stats using Framer Motion
- **Persian Typography** — Optimized with Vazirmatn font
- **Fast Performance** — Built with Next.js and Turbopack
- **Accessible** — RTL support, smooth scrolling, and clean code

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + custom CSS
- **Animations**: Framer Motion
- **Font**: Vazirmatn (CDN)

## 📦 Installation & Run

Clone the repository and install dependencies:

```bash
git clone https://github.com/AminTheCodeReaper/Hackathon-Front.git
cd Hackathon-Front
npm install
npm run dev --turbo
Open http://localhost:3000 to see the result.
```
